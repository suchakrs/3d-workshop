Fundamentals of 3D Development

Interactive half-day workshop. Each example is a standalone explorable playground — open it, tweak the controls, read the matching doc in workshop-docs/. The examples build up one evolving scene, ending in a 1 million-point WebGPU point cloud.

  1. 3D space & transforms — coordinate space, position/rotation/scale, parent-child
    • 01-2 · Transform matrices — drag a gizmo, read the live 4×4 matrix
  2. Modeling & topology — vertices, faces, BufferGeometry, normals, winding
  3. Materials — Basic → PBR Standard/Physical, maps, transparency
  4. Lighting — ambient/directional/point/spot, shadows, IBL
  5. Cameras — perspective vs ortho, FOV, near/far, frustum
  6. three.js (vanilla) — the full assembled scene + render loop
  7. React Three Fiber — the same scene, declaratively
  8. Point cloud (WebGPU + TSL) — 1M points from a 3D scan, high-performance rendering

Docs & setup: workshop-docs/00-overview.md · Point-cloud deep dive (take-home): workshop-docs/08-pointcloud-webgpu-tsl.md
